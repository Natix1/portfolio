Got Questions?

Why not sell this as an obfuscated product for general use by groups?

I oppose obfuscation, therefore you're not able to buy a obfuscated version like you would with most tech products. You'll have to talk with me on discord directly if you're interested - XenPOS is only be sold to reputable people, for codebase security reasons.

How much does XenPOS cost?

The price is 40,000 R$ or 152 US$ (via paypal). It is not negotiable, to make it fair to any current and future customers.

Do I have to pay for updates separately?

No. You'll receive a structure containing a Package, which will automatically update your local copy of XenPOS whenever an update is released, unless you modify the source. If you choose to do so, I'll be able to provide update files alongside a git commit history which you will merge into your modified XenPOS copy yourself.

Is this project modular?

Every single piece of code in XenPOS is a ModuleScript, which allows you to hook into and modify a lot of XenPOS features without touching the actual codebase at all.

Would you be interested in a partnership?

I might be, depending on the context. Don't be afraid to reach out.

Is this project maintained?

XenPOS is maintained, although features are mainly added per request. Bug fixes are pushed out ASAP after one has been found.
